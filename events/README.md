CDI Events Example
==================

This example demonstrates how to use CDI events.


Build it
--------

```shell
gradle -t build
```


Run it
------

```shell
gradle -t run
```


Develop it
----------

```shell
gradle -t start
```

Don't forget to stop running application at the end of the day:

```shell
gradle stop
```


Distribution
------------

```shell
gradle installDist
```

Run it:

```shell
build/install/events/bin/events
```
